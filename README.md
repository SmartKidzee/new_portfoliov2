## ✦ About This Project

> _A cinematic, dark-themed portfolio that blends AI-first thinking with polished frontend craft — built to impress, designed to perform._

**Shreyas J — Portfolio v2** is a modern, immersive single-page portfolio featuring cinematic transitions, liquid-metal UI effects, dynamic blog integration, and interactive skill showcases. Every pixel is intentional. Every animation tells a story.

<br>

### ✨ Highlights

- 🎬 **Cinematic Hero** — HLS video background with GSAP-powered reveal animations and dynamic role cycling
- 🌊 **Liquid Metal UI** — Custom WebGL shader-powered buttons and interactive elements with metallic reflections
- 📊 **Skills Showcase** — Interactive radar-style skill visualization with animated progress indicators
- 📝 **Dynamic Blog System** — Supabase-backed blog with AI reading assistant, like system, and social sharing
- 🕰️ **Interactive Timeline** — Scroll-driven sticky timeline with GSAP animations and phase cards
- 💬 **Testimonial Marquee** — Smooth infinite-scroll testimonial chips with responsive layout

## 🏗️ Project Structure

```
new_portfoliov2/
├── app/
│   ├── api/              # Server endpoints (contact, blog APIs)
│   ├── blogs/            # Dynamic blog routes
│   ├── privacy/          # Privacy policy page
│   ├── terms/            # Terms of service page
│   ├── globals.css       # Design system & global styles
│   ├── layout.tsx        # Root layout with fonts & analytics
│   ├── page.tsx          # Home page with structured data
│   ├── sitemap.ts        # Dynamic sitemap generation
│   └── robots.ts         # Robots.txt configuration
├── components/
│   ├── ui/               # Reusable UI primitives
│   │   ├── liquid-metal  # WebGL shader button components
│   │   ├── timeline      # Scroll-driven timeline
│   │   └── warp-bg       # Perspective beam grid
│   ├── portfolio-shell   # Main single-page layout (~1300 lines)
│   ├── skills-showcase   # Interactive skill radar section
│   ├── blog-ai-assistant # AI-powered reading companion
│   ├── testimonials      # Marquee testimonial section
│   └── ...               # Blog carousel, counters, HLS video
├── content-data/         # JSON content layer
│   ├── projects.json     # Featured work entries
│   ├── skills.json       # Skill categories & progress
│   ├── timeline.json     # Growth timeline entries
│   └── ...               # About, education, stats, etc.
├── lib/                  # Utilities, content loaders, helpers
├── public/               # Static assets (logo, OG image, hero)
└── supabase/             # Database schema & client config
```

## 🚀 Quick Start

> **Prerequisites:** Node.js 18+, npm or yarn

```bash
# Clone the repository
git clone https://github.com/SmartKidzee/new_portfoliov2.git

# Navigate to the project
cd new_portfoliov2

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Fill in your Supabase, email, and API keys

# Start development server
npm run dev
```

<div align="center">

The app will be running at **`http://localhost:3000`** 🎉

## 🌐 Environment Variables

Create a `.env.local` file based on `.env.example`:

| Variable | Purpose |
|:---|:---|
| `NEXT_PUBLIC_SUPABASE_URL` | Supabase project URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase anonymous key |
| `SMTP_HOST` / `SMTP_PORT` / `SMTP_USER` / `SMTP_PASS` | Email delivery via Nodemailer |
| `WEB3FORMS_KEY` | Backup contact form delivery |
| `OPENROUTER_API_KEY` | AI blog assistant (OpenRouter) |
